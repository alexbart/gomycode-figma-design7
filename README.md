# Design 7 – Responsive Card Layout (Bootstrap 4)

## Project Overview

This project is a responsive card layout built using **Bootstrap 4.6**.  
It demonstrates proper use of the Bootstrap grid system to create a clean, responsive multi-column layout.

The design adapts across different screen sizes using Bootstrap’s breakpoint system.

---

## Features

- Responsive grid layout
- Bootstrap 4.6 card components
- Mobile-first design
- Proper column breakpoints
- Clean semantic HTML structure

---

## Technologies Used

- HTML5
- Bootstrap 4.6
- CSS (Bootstrap utility classes)
- jQuery & Popper.js (Bootstrap dependencies)
